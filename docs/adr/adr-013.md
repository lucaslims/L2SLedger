# ADR-013 — LGPD e Proteção de Dados Financeiros no L2SLedger

## Status

Aprovado

## Contexto

O **L2SLedger** processa e armazena **dados financeiros e dados pessoais**, estando sujeito às disposições da **Lei Geral de Proteção de Dados (LGPD)**.

É necessário estabelecer diretrizes claras para:

* Tratamento de dados pessoais
* Proteção de dados financeiros
* Minimização de riscos legais e operacionais

---

## Decisão

Adotar uma política explícita de **proteção de dados alinhada à LGPD**, baseada nos princípios de minimização, segurança e responsabilidade.

---

### 1. Classificação de Dados

* **Dados pessoais**:

  * Identificadores de usuário
  * E-mail

* **Dados financeiros**:

  * Lançamentos
  * Saldos
  * Histórico financeiro

---

### 2. Princípios LGPD Aplicados

* Finalidade: dados usados apenas para operação do sistema
* Necessidade: coletar apenas dados estritamente necessários
* Segurança: proteção técnica e organizacional
* Transparência: possibilidade de auditoria

---

### 3. Segurança dos Dados

* Criptografia em trânsito (HTTPS)
* Criptografia em repouso para banco de dados
* Controle de acesso restrito no backend
* Logs sem exposição de dados sensíveis

---

### 4. Retenção e Exclusão

* Política clara de retenção de dados
* Exclusão de dados mediante solicitação válida
* Dados financeiros respeitam obrigações legais de retenção

---

### 5. Responsabilidades

* Backend é o **controlador técnico** dos dados
* Firebase atua apenas como provedor de identidade
* Nenhum dado sensível é compartilhado com terceiros sem justificativa legal

---

## Consequências

### Positivas

* Redução de risco legal
* Maior confiança do usuário
* Base sólida para compliance futuro

### Negativas

* Maior rigor em processos e desenvolvimento

---

## Alternativas Consideradas

* Ausência de política explícita de LGPD (rejeitado)
* Tratamento informal de dados sensíveis (rejeitado)

---

## Conclusão

A adoção de uma política clara de **LGPD e proteção de dados financeiros** é essencial para a credibilidade, sustentabilidade e conformidade legal do L2SLedger.
Esta decisão estabelece as bases para o tratamento responsável dos dados, alinhado às melhores práticas e exigências legais.
